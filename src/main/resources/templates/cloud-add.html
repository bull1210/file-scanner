Register OAuth App

Create an app in the Azure Portal → App registrations and add the redirect URI below. Grant the Files.Read.All, User.Read, and offline_access delegated permissions.

  • Go to portal.azure.com → App registrations
  • New registration → set Redirect URI (Web) to the URI above
  • API permissions → Add Files.Read.All, User.Read (delegated)
  • Certificates & secrets → New client secret → copy value

Create credentials in Google Cloud Console → APIs & Services → Credentials and add the redirect URI below. Enable the Google Drive API and Google People API for the project.

  • Go to console.cloud.google.com → Credentials
  • Create Credentials → OAuth client ID → Web application
  • Add the Redirect URI above under "Authorised redirect URIs"
  • Copy Client ID and Client Secret
  • Enable Google Drive API and Google People API in Library
